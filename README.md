# Sweet Shop - Frontend UI

A modern e-commerce frontend for a Japanese-inspired sweet shop, built with React.

## Features

- Clean and modern UI design
- Responsive layout
- Product grid display
- Search and filter functionality (UI only)
- User authentication buttons (UI only)
- Floating help button

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

The app will open at [http://localhost:3000](http://localhost:3000)

## Project Structure

```
src/
  components/
    Header.js          # Header with logo, title, and auth buttons
    SearchFilter.js    # Search bar and filter button
    ProductCard.js     # Individual product card component
    ProductGrid.js     # Grid of product cards
    Footer.js          # Footer component
    HelpButton.js      # Floating help button
  App.js              # Main app component
  App.css             # Main app styles
  index.js            # Entry point
  index.css           # Global styles
```
